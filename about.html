This Website is built with Html, CSS, Javascript, Bootstrap and Rapid Api.
In this website you can search the Weather report of the desired city.
You can search in the search option in Navigation bar.
Parameters
lat (required) - latitude of desired location.
lon (required) - longitude of desired location.
— or —
zip (required) - 5-digit Zip code (United States only)
— or —
city (required) - city name.
state (optional) - US state (for United States cities only).
country (optional) - country name.
